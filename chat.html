Security notes (MVP)

  • Encryption is done in your browser using NaCl box (X25519 + XSalsa20-Poly1305) and Double Ratchet for forward secrecy.
  • The relay server can’t read messages, but it can still see metadata (who/when).
  • Verify the Safety code with your peer before trusting them.
  • Double Ratchet and X3DH provide Signal-tier cryptographic security for all sessions.
