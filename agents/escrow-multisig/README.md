# Multisig Escrow Example (scaffold)

This folder contains a starter scaffold for a multisig-based escrow example using SPL tokens. It is intentionally minimal — use it as a developer example to test flows locally.

Implemented scripts (demo only):

- `create-multisig.js` — create a multisig (default 3 signers, 2-of-3 threshold). Writes keys to `agents/escrow-multisig/data`.

- `fund-escrow.js` — fund the multisig's associated token account from the payer's token account.

- `release-escrow.js` — release tokens from the multisig to a recipient; uses the first `threshold` signer keypairs to sign the transfer.

Usage examples (devnet):

1. Create multisig (requires `MINT` and `KEYPAIR_PATH` in env):
   KEYPAIR_PATH=~/payer.json MINT=<mint> node agents/escrow-multisig/create-multisig.js

2. Fund multisig (transfer tokens from payer to multisig ATA):
   KEYPAIR_PATH=~/payer.json MINT=<mint> AMOUNT=1000000 node agents/escrow-multisig/fund-escrow.js

3. Release from multisig to recipient (uses signer key files generated by create-multisig):
   KEYPAIR_PATH=~/payer.json RECIPIENT=<recipient_pubkey> AMOUNT=1000000 node agents/escrow-multisig/release-escrow.js

Important: this is a developer demo. Do not use in production without proper audits and program-controlled escrow accounts.
